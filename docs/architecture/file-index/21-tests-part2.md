# Tests (part 2) — File Index
> Scope: `Tests/RepoPromptTests/{MCP,Services,WorkspaceContext,Workspaces}` + root-level tests. 87 files.

## Subsystem role
This half of the XCTest suite verifies RepoPrompt CE's server-side and storage-layer machinery: the MCP server (socket transport, JSON-RPC bridge, tool execution contracts/watchdogs, agent-run lifecycle, worktree tooling, CLI installers, config export), the FileSystem/VCS services (watcher ingress, ignore rules, git worktree create/merge/branch flows), the WorkspaceContext store (file/codemap catalogs, store-backed search + admission control, path matching, selection slices/persistence), workspace checkout refresh, and the CE-owned JSON-only persistence for chat history, presets, and settings. Tests favor deterministic fixtures, source-text invariant guards, and DEBUG-gated integration harnesses (socketpair MCP, real git repos) to lock down concurrency, recovery, and routing correctness.
